# Play-Zone-API
RESTful API for PLAY-ZONE, a gaming social network platform.

This is the backend API for PLAY-ZONE, a full-stack social platform for gamers. It enables users to:

- Register and authenticate securely
- Create and edit profiles with avatars, bios, and preferences
- Publish posts, reviews, and guides
- Follow other users and interact with favorite content

Built with Node.js, Express, and MongoDB, the API follows RESTful conventions and supports modular expansion for future features.

### Tech Stack
- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication
- Multer + Cloudinary for image uploads
